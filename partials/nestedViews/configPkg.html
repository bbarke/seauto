Framework

This is where some of the configurable SeAuto framework reside

Both of these files are required to use the StoryContext feature. See Story Context: Setup for more information on how to create and use the objects.

  • StoryContext
  • StoryContextProvider

Selenium

Inside of the Selenium package, DriverProviderProducer and ConfigurableDriverProvider are found. These are here to maintain the lifecycle of DriverProvider, which provides a thread safe WebDriver. They are extracted in this way so specific implementations of the WebDriver can be created.

Defaults can be used and these files can be removed by setting beans.xml to: 

<beans
  xmlns="http://java.sun.com/xml/ns/javaee"
  xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
  xsi:schemaLocation="
  http://java.sun.com/xml/ns/javaee
  http://java.sun.com/xml/ns/javaee/beans_1_0.xsd">

  <alternatives>
    <class>com.partnet.automation.selenium.DefaultDriverProviderProducer</class>
  </alternatives>
</beans>
